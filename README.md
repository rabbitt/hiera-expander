# hiera-expander

Hiera backend that causes each data source to be expanded into multi-level sources.

## Installation

Add this line to your application's Gemfile:

    gem 'hiera-expander'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hiera-expander
